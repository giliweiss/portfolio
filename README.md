# Gili Weissberg · Portfolio

Personal portfolio - full stack developer, CS graduate from Ben-Gurion University.

**Live:** [giliweiss.github.io/portfolio](https://giliweiss.github.io/portfolio)

---

### Run locally

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000).

### Build

```bash
npm run build
```

Builds to the `build` folder (deployed via GitHub Actions).
